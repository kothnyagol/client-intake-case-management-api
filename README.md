# Client Intake & Case Management API

## Project Description
This is a Django + Django REST Framework API for a law firm to manage clients, cases, and documents. It allows lawyers to:

- Add, view, update, and delete clients.
- Create, view, update, and delete cases.
- Assign cases to specific lawyers.
- Upload and manage documents linked to cases.
- Authenticate users (lawyers/admins).

## Technologies Used
- Python 3
- Django 4.2
- Django REST Framework
- SQLite (default DB)
- Git & GitHub
